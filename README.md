# 📔 日历日记

一个简单的网页版日记应用，支持按日历记录每日心情。

## 功能特点

- 📅 日历视图浏览
- ✍️ 记录每日日记
- 📋 日记列表管理
- 💾 数据本地存储

## 在线访问

访问地址：https://aiweisoft.github.io/calendar-diary

## 使用说明

1. 点击日历上的日期选择要写日记的日子
2. 在文本框中输入日记内容
3. 点击"保存日记"按钮保存
4. 点击"日记列表"查看所有已写日记

## 技术栈

- HTML + CSS + JavaScript
- GitHub Pages 部署
- GitHub Actions 自动部署
